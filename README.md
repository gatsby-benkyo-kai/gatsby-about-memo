## セットアップ手順

gatsby-cliをインストール
```
$ npm install --global gatsby-cli
```
バージョンの確認
```
$ gatsby --version

Gatsby CLI <バージョン>
```
新規プロジェクトの作成
```
gatsby new <プロジェクト名>
```
プロジェクト名のフォルダへ移動
```
$cd <プロジェクト名> 
```
gatsby developコマンドでサーバー起動
```
gatsby develop
```
下記の通り起動に成功したら
http://localhost:8000/
へアクセス
